# ChatGPT Simple Playground

Are you looking for a way to engage your users and keep them coming back to your website? Look no further than our amazing ChatGPT Simple Playground.

Our simple yet effective plugin allows users to play, interact and engage with your website like never before. With the ability to customize the number of requests per day, you can tailor the experience to fit your specific needs.

ChatGPT Simple Playground is the perfect addition to any website or business that wants to increase engagement and improve user experience. 

Summary

*   Set limit to how many times the website can send a request to ChatGPT API
*   Make your website more engaging and interactive
*   Test ChatGPT API

== Frequently Asked Questions ==

= Is it free? =

Yes the plugin is free but you will need to use your own openai API key, which has a cost to it.

= Where can I get the API? =

Go to [OpenAI](https://openai.com/blog/openai-api "OpenAI API") website, log in and go to My account>API. 

= Do you support Azure OpenAI too? =
Yes, please go to [Azure OpenAI API]("https://azure.microsoft.com/en-us/pricing/details/cognitive-services/openai-service/") to get your API key and review their privacy policy.


== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `ChatGPT Simple Playground` and click 'Install Now' or click on the `upload` link to upload `kh-chatgpt-frontend-fun.zip`
4. Click on `Activate plugin`
